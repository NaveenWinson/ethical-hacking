Recover files from shortcut virus in Mac OS X

Let us see how to recover the files from a pen drive that has been affected by the shortcut virus. This method can only be done on Mac OS X


1. This is very easy one. Right Click on the pendrive with the shortcut virus.



2. Click duplicate option in the right click menu.



3. Thats it in the duplicate folder you can see all the files that as present in that pendrive
