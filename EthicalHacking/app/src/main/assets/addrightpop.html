Adding entry to the Right Click Menu

Let us see how to add entry in the right click pop-up menu.



1. Open regedit i.e Registry Editor in Windows PC



2. Expand HKEY classes root



3. Then go to txtfile -> shell



4. Add a new key to shell



5. Type the value that you would like to add it to the right right click pop-up menu.



6. Now create a new text file and right click, you would see the value that you entered
