Basics and Requirements

There are basics you should know before going through the tutorials. The basics includes the requirement that your PC must have to do the tutorials.


The most important thing is that you have Kali Linux OS installed in your PC. If you are using a virtual machine for Kali Linux, please do make sure that you have a wireless interface card that has the capability of injection.


The first five tutorial will be dealing with Kali Linux and simple hacking. And the next five will be in Windows which would be more ethical.


Before proceeding with the tutorials, make sure that you are familiar with kali linux GUI and also with terminal. For each tutorial there is a video embeded if you have any doubts you can watch it.
