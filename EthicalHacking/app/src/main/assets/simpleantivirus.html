Create a Simple Antivirus

Let us see how to make a simple antivirus in Windows.



1. Open the directory where the virus is present. Create a new text document



2. Type
@echo off
if exist virus.bat goto infected
:infected
echo virus found
del virus.bat
echo virus deleted



3. Save this text document with extension .bat



4. The script will delete any bash script that is present in its directory as the virus are made using the bash script.Hence when you execute the script it will delete the virus in its directory.
